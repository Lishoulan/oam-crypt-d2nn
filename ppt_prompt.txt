# 幻灯片 8: v8 PolarHNN 核心创新概览

|------|----------|------|------|

# 幻灯片 9: 创新 1 - PolarConv 极坐标卷积

标题: PolarConv: 极坐标空间的方位角卷积

物理动机:
- OAM 模式 exp(il*theta) 在方位角 theta 方向有 l 个周期
- l=25 时 25 圈螺旋, 笛卡尔 3x3 卷积窗口无法覆盖
- 必须在极坐标 (r, theta) 空间沿 theta 方向做 1D 卷积

实施方案流程:

输入 (B, C, H, W)
  |-- grid_sample 笛卡尔 -> 极坐标
  v
极坐标 (B, C, n_r=32, n_theta=96)
  |-- theta 方向 Conv1d kernel=7 (深度可分离, groups=C)
  |-- r 方向 Conv1d kernel=3
  v
极坐标 (B, C, n_r=32, n_theta=96)
  |-- grid_sample 极坐标 -> 笛卡尔
  v
输出 (B, C, H, W) = x + scale * (x_out - x)

关键设计:
- 残差连接 + init_scale=0: 训练初期等于恒等, 后期 scale 增长
- n_r=32, n_theta=96: 3072 采样点 (vs 270x270=72900 笛卡尔点)
- 参数量仅增加 ~76K (深度可分离, groups=C)

物理意义:
- theta 方向 1D 卷积 = 沿方位角做谐波滤波
- 天然对应 OAM 拓扑荷 l 的方位角谐波结构
- kernel=7 覆盖 2pi 的 7/96 = 7% 范围, 匹配 l<=25 的高阶谐波

# 幻灯片 10: 创新 2 - OAM-FDD Loss 频域正交损失

标题: OAM-FDD Loss: 通道间频域正交约束

物理动机:
- 10 通道 OAM 模式在中心 216x216 完全空间重叠
- 单纯依赖 D2NN 隐式分离困难
- 显式约束通道间频域正交性, 化解"同位置重叠"串扰

实施步骤:

1. 对每通道沿 W 维度 (方位角方向) 做 FFT
2. 排除 DC bin (图像能量大部分在 DC, 导致所有通道相关性接近 1)
3. 排除 Nyquist bin
4. L2 归一化每个通道的频域向量
5. 计算通道间点积矩阵 R_ij = |<Y_i, Y_j>|
6. 损失 = R_offdiag.mean() (理想正交时为 0)

关键修复:
- 必须 .float(): cuFFT 在 half 精度下不支持非 2 幂维度 (如 1080)
- 必须排除 DC bin: 否则所有通道相关性都接近 1 (DC 主导)
- 必须 L2 归一化: 让相关性成为"形状相似度"而非"能量相似度"

物理意义:
- 沿 W 维度 FFT = 沿方位角做谐波分解
- 不同 OAM 通道应在不同谐波 bin 集中能量
- 通道间频域相关性 -> 0, 意味着 OAM 模式在频域正交

# 幻灯片 17: SLM Lee Hologram 物理编码

标题: Lee 离轴全息编码 - 复振幅到纯相位

问题:
- v8 cipher 是复振幅 U_cipher (含振幅 + 相位)
- SLM 只能加载纯相位 (phase-only)
- 需要将复振幅编码为纯相位格式

Lee Hologram 公式:
H(x, y) = arg( R + U_cipher(x, y) * exp(i * 2*pi*f0*x) )

其中:
- R = max|U| (参考光偏置, 0 dB 满量程)
- f0 = 0.125 cyc/pix (载波频率, 8 pix/cyc)
- arg() 取相位

重建原理:
- SLM 加载 H -> 4f 傅里叶变换 -> 频域出现 0 级和 +/-1 级
- +1 级位于 f0 偏移处, 包含完整 U_cipher
- 圆形滤波窗口提取 +1 级 -> 逆 FFT -> 恢复 U_cipher

SLM 物理参数:

| 参数 | 值 |
|------|----|
| SLM 型号 | Holoeye PLUTO-2.1 |
| 分辨率 | 1920x1080 |
| 像素尺寸 | 8.0 um |
| 相位级数 | 8-bit (256 级) |
| 波长 | 532 nm |
| 载波周期 | 8 pix/cyc |
| 频偏 f0 | 15.6 cyc/mm |

# 幻灯片 19: 端到端物理链路验证

标题: Lee Hologram + 4f + D2NN 端到端物理链路

完整链路:

明文 -> OAM 加密 + RPP -> 复振幅 U_cipher
  -> Lee 编码 -> SLM 8-bit 灰度加载
  -> 4f 傅里叶变换 (FFT 到频域面)
  -> +1 级圆形滤波 (中心 f0=0.125 cyc/pix)
  -> 逆 FFT 回物面
  -> D2NN 解密
  -> 重建 MNIST

实验矩阵 (3 偏置 x 3 滤波窗口 = 9 配置):

| 偏置 | 滤波 135 pix | 滤波 270 pix | 滤波 540 pix |
|------|---------------|---------------|---------------|
| 0 dB | 11.23 dB | 11.48 dB | 15.21 dB |
| 6 dB | 11.21 dB | 11.29 dB | 12.04 dB |
| -3 dB | 11.25 dB | 11.89 dB | 18.54 dB |

最佳配置: -3 dB 偏置 + 540 pix 滤波窗口

关键结果:
- 数字 baseline (SLM-aware): 24.93 dB
- Lee + 4f + D2NN 端到端: 18.54 dB
- 损耗: -4.9 dB

物理解释:
- RPP 随机相位把频谱展宽到几乎全频
- 滤波窗口必须足够大 (540 pix = 50% 频域) 才能保留大部分信号
- 过调制 (-3 dB) 牺牲线性但提升 +1 级能量, 综合最优

结论: Lee 全息图 + 4f + D2NN 端到端物理可行, 18.54 dB PSNR 视觉清晰

# 幻灯片 21: 物理边界分析

标题: 10 通道 oam_overlap 的物理上限

现象:
- v5/v6/v7/v8 四个版本, 10 通道都在 11-14 dB 区间
- 算法创新 (ChannelAttention, Curriculum, OAMFreqFilter, PolarConv) 无法突破

物理原因:
1. 10 个 OAM 模式在中心 216x216 完全空间重叠
2. D2NN 4 层相位 + 2-3 层 Refine UNet, 难以在 z in [0.05, 0.95] m 内完全正交分离
3. 频域 OAM 谐波 bin 距离 = 1/1080 = 0.001 cyc/pix, 谱分辨率限制

信息论估算:
- 10 通道在 1080x1080 域, 自由度数 = 10 * 1080^2 = 1.17 * 10^7
- 4 层 D2NN 相位掩模自由度数 = 4 * 1080^2 = 4.67 * 10^6
- 信息论下限: 1.17e7 / 4.67e6 = 2.5x 信息压缩
- 对应 PSNR 衰减到 ~14 dB

结论: 2-3 层 D2NN 的信息论容量不足以完全分离 10 通道 oam_overlap

备注: 这是物理/信息论层面的限制, 不是工程调优能解决的。需要新的物理范式 (如 v8.1 多尺度 OAM 解码) 或增加 D2NN 层数 (需要更大显存)。

# 幻灯片 27: 关键公式汇总

标题: 核心物理与数学公式

OAM 模式:
U_OAM^l(r, theta) = A(r) * exp(i * l * theta)

多通道密文 (oam_overlap):
U_cipher(x, y) = [Sum_{j=1}^{N} M_j(x, y) * exp(i * l_j * theta)] * exp(i * phi_RPP(x, y))

Lee Hologram 编码:
H_Lee(x, y) = arg(R + U_cipher(x, y) * exp(i * 2 * pi * f0 * x))

4f 重建 +1 级:
U_recovered = F^{-1}{circ_r(F{exp(i * H_Lee)})} * 2R

PSNR_C (中心 216x216):
PSNR_C = 10 * log10(1 / MSE(I_pred, I_true))

OAM-FDD Loss:
L_FDD = (1 / n_pairs) * Sum_{i<j} |<Y_i, Y_j>| (排除 DC bin, L2 归一化)

PolarConv 残差:
x_out = x + scale * (PolarConv(x) - x)

备注: 所有公式均使用国际单位制。OAM 模式中 A(r) 是径向振幅分布, l 是拓扑荷 (整数), theta 是方位角。Lee Hologram 中 R 是参考光偏置, f0 是载波频率。4f 重建中 circ_r 是圆形低通滤波器, F 和 F^{-1} 分别是正逆傅里叶变换。
